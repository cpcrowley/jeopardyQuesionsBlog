Quick Start Choose a topic from the dropdown menu at the top of the page.

Why this site? I created this site because I had questions about Jeopardy.

Notes Each topic page ends with some notes and observations about the data.

Data source The data used here are taken from the excellent Jeopardy site J! Archive.

Additions and corrections If there are any statistics you want the site to compute send me an email and I will try to add it.

Technical details I collected the data by scraping the HTML from j-archive and converting it to a JSON file which the server reads in and uses to create the tables. Everything is written in JavaScript. Several node.js programs are used to collect and summarize the data. The code is on Github here.
